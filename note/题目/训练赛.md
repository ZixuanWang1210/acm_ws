# B - xz的字符串

xz现在有 $n$ 个字符串 $S_1,...,S_n$ 

定义字符串 $S_i$ 的权值 $w_i$ 为 $S_i$ 与其前面的字符串（即 $S_1,…,S_{i-1}$ ）中所有最长公共前缀的最大长度

有 $2$ 种操作：

1. `1 x` ：将字符串 $S_x$ 翻转
2. `2 x`：询问字符串 $S_x$ 的权值

有 $q$ 次操作，对于每次操作2输出查询结果

**输入格式**

第一行一个正整数 $n$ $(1\le n \le 10^5)$

(|S| \le 100)$

下一行一个整数 $q$ $(1\le q \le 10^5)$

接下来 $q$ 行，每行包含两个正整数 $op ,  x$ $(op=1,2、1\le x \le n)$

数据保证操作1的数量不超过10次

**输出格式**

每行一个正整数表示询问2的结果

**输入样例**

```
3
abc
aadc
aaaaaa
3
2 3
1 2
2 3
```

**输出样例**

```
2
1
```
